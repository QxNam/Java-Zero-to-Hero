# Bài 2: Cấu trúc điều khiển (Control Structures)
## I. Tổng quan
Về cơ bản nhất, một chương trình là một danh sách các lệnh. Cấu trúc điều khiển là các khối lập trình có thể thay đổi đường đi mà chúng ta thực hiện thông qua các lệnh đó. \
Có ba loại cấu trúc điều khiển:
- Các nhánh điều kiện, được sử dụng để lựa chọn giữa hai hoặc nhiều đường dẫn: 
  + `if`, `else`, `else if`
  + toán tử ba ngôi
  + switch-case
- Vòng lặp được sử dụng để lặp qua nhiều giá trị/đối tượng và chạy lặp đi lặp lại các khối mã cụ thể:
  + for
  + while
  + do while
- Các câu lệnh rẽ nhánh, được sử dụng để thay đổi luồng điều khiển trong vòng lặp:
  + break
  + continue
